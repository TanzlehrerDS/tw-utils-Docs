# tw-utils Documentation

Welcome to the documentation of **tw-utils** — tools and helpers for the
browser game *Tribal Wars*.

## Tools

- [Off-Planner](off-planner/index.md) — Plan offensive attacks
- [AG-Planner](ag-planner/index.md) — Coordinate AG spam
- [ZWC-Planner](zwc-planner/index.md) — Calculate synchronized waves
- [Discord Bot](discord-bot/index.md) — Tribe integration in Discord

## About tw-utils

tw-utils is a hobby project. The source code is available on
[GitHub](https://github.com/Tanzlehrer/tw-utils).
