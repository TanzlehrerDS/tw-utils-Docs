# Tab 2: Targets

In Tab 2 you list the villages you want to attack with your noblemen.

## Adding targets

Click the "Add targets" button and paste the coordinates of the target villages.

## Runtime priority

You can then assign a runtime priority to each target. It determines the order
in which the planner tries to assign noblemen to a target.

## Overview

As in Tab 1, a search field helps you keep an overview as the target list
grows.
